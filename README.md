# Discord Token Checker
* Python : 3.6+
***
## Installing Requests:
```console
$ python -m pip install requests
```
***
## Usage:
1. Create file `tokens.txt` and enter your tokens (1 token - 1 line).
2. `Checker.py` and tokens.txt must be in the same folder.
3. Run `checker.py`.
```console
$ python checker.py
```
